Planetodo

惑星と太陽系をイメージした、タイマー・タスク管理アプリです。
ポモドーロテクニック(※1)を踏襲し、
1セットを地球の公転1周にたとえて表現しました。

  • 作業時間・休憩時間を設定できるタイマー機能(※2)
  • タスクリスト機能
から構成されています。

※1 ポモドーロテクニック
作業時間と休憩時間を合わせたセットを1単位(1ポモドーロ)としてタスクを行う手法。作業時間中は集中を切らさないこと、セット中はタスクの内容を変えないことが鉄則。
作業時間25分が目安とされるが、集中力の続く時間であれば長さは自由。

※2
テスト・審査のため、本来ではあり得ない短さの時間の選択肢を故意に複数入れています。

Timer

タイマーの盤面です。

作業時間・休憩時間 2色の太い円周
選択中のタスクなど 文字
現在時点 白い小円(地球)
経過時間 白い細い軌道(公転軌道)
現在の周回数 黄色円(太陽)の数字
で表します。

Settings

作業時間(Work period)・休憩時間(Break period)を設定します。
作業 + 休憩で1セット = 公転1周 = 1 Orbit とし、
その時間配分を選ぶことでTimerに時間配分を表す盤面が表示されます。
(※2 参照)

Status

タイマー作動後に現在の状況を、
「WORK」「BREAK」「FINISHED!」
のいずれかで表示します。

Switch

タイマーのスイッチです。
タスクの選択・時間配分の選択が揃った状態で【start】を押すと、
時間の計測が始まります。
【restart】で時間の測りなおし、
【reset】でその周のタイマーをリセットします。
スタート時・休憩開始時・タイマー終了時に通知音が鳴ります。

Todo List

やるべきタスクのリストです。
テキストボックスに入力して【Add】または Enter押下で、
タスクをTodo Listに追加します。
これから取り組むタスクを select欄のラジオボタンで選択します。
【delete】で削除、【done】でDone Listへ追加、
【keep】でKeep Listへ追加することができます。

Done List

作業を終え、Todo Listから移動したタスクのリストです。
Orbit は終えたセット数、
Total worked min は合計作業時間(分)を表します。
Done Listに追加されたタスクや合計作業時間などは
成果としてツイートすることができます。

Keep List

Todo Listによく追加するタスクを保存しておくためのリストです。
Todo ListやDone Listから【keep】でkeep Listに入れて保存、
【Todo】でTodo Listへの再追加ができます。
ほとんどの内容は更新ボタンでクリアされますが、Keep Listに加えておくと 次回以降も自動でリロードされ、簡単にTodo Listに加えることができます。
【clear】でKeep Listの内容を全削除します。
