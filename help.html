Planetodo

惑星と太陽系をイメージした、タイマー・タスク管理アプリです。
ポモドーロテクニック(※)を踏襲し、
1セットを地球の公転1周に模しました。
作業時間・休憩時間を設定できるタイマー機能、タスクリスト機能から構成されています。

※ ポモドーロテクニック
作業時間と休憩時間を合わせたセットを1単位(1ポモドーロ)としてタスクを行う手法。作業時間中は集中を切らさないこと、セット中はタスクの内容を変えないことが鉄則。
作業時間25分が目安とされるが、集中力の続く時間であれば長さは自由。

Timer

作業時間・休憩時間 2色の太い円周
選択中のタスクなど 文字
現在時点 白い小円(地球)
経過時間 白い細い軌道(公転軌道)
現在の周回数 黄色円(太陽)の数字

で表します。

Settings

時間の設定と計測・セット数のカウントができます。
作業 + 休憩で1セット( = 公転1周)とし、その時間配分を選ぶことで各時間が色分けされて表示されます。

Status

タイマー作動後に現在の状況を
WORK / BREAK / FINISHED!
のいずれかで表示します。

Switch

タスクの選択・時間配分の選択が揃って【start】すると、時間の計測が始ります。
【restart】で時間の測りなおし、
【reset】でその周のタイマーをリセットします。
スタート時・休憩開始時・タイマー終了時に通知音が鳴ります。

Todo List

Todo List内テキストボックスに入力して【Add】または Enter でタスクをTodo Listに追加します。
これから取り組むタスクを select欄のラジオボタンで選択します。

Done List

Todo Listで終わったタスクは【Done】でDone Listに移動します。
revolution は終えたセット数、
total work min は合計作業時間(分)です。
Done Listに追加されたタスクや合計作業時間などは成果としてツイートすることができます。

Keep List

頻繁にTodo Listに追加するタスクはkeep Listに入れて保存・【Todo】でTodo Listへの再追加ができます。
ほとんどの内容は更新ボタンでクリアされますが、Keep Listに加えておくと次回以降もリロードされ、簡単にTodo Listに加えることができます。
【clear】でKeep Listの内容を削除します。
